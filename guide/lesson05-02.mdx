# Deberes

Modifica tu componente `info-box` para que reciba el texto que se muestra dentro del cuadrado verde como una propiedad.
En caso de no recibir ningún texto, debes asegurarte de que la altura no se vea afectada.

Crea un componente `button-symbol`, que recibirá un símbolo. Este símbolo puede ser uno de los siguientes ocho: ("←", "\*", "+", "-", "/", ".", "=") o bien un número del 0 al 9.

Al hacer clic, el componente emitirá un evento llamado `operation`, donde en el campo detail enviarás el símbolo recibido.

Debes verificar que el símbolo introducido es uno de los indicados; en caso contrario, el componente no debe renderizar nada.

> Recuerda: debes crear también un stories y un README bien documentado para cada componente.
