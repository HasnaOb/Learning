Diferencia entre Funciones Tradicionales y Arrow Functions

Este ejemplo muestra cómo el valor de this varía entre funciones tradicionales y arrow functions.

Función Tradicional


const objTradicional = {
  nombre: "Pedro",
  mostrarThis: function() {
    console.log(this);
  }
};
objTradicional.mostrarThis(); // this = objTradicional

Arrow Function


const objArrow = {
  nombre: "Lucía",
  mostrarThis: () => {
    console.log(this);
  }
};
objArrow.mostrarThis(); // this = contexto global (window o undefined en strict)
