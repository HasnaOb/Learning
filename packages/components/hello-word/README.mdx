# HelloWord component

This component will display a screen with an initial text

## Properties

Component can be configured via these properties:

| Name           | Type    | Description                        | Default | Private |
| -------------- | ------- | ---------------------------------- | ------- | ------- |
| `translations` | Object  | The translations are stored inside | null    | true    |
| `hasSymbol`    | Boolean | Show icon or not                   | false   | false   |
| `listString`   | Object  | List for elements                  | []      | false   |
| `description`  | String  | Description for example            | ""      | false   |

## Events

| Name                          | Description            | Detail | Composed | Bubbles |
| ----------------------------- | ---------------------- | ------ | -------- | ------- |
| `{HelloWord.events.continue}` | By clicking the button | Object | true     | true    |

## Usage

```html
<hello-word></hello-word>
```
