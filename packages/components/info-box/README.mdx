# InfoBox component

This component renders a box that displays a text message inside.  

## Properties

The component can be configured via the following properties:

| Name           | Type   | Description                               | Default | Private |
| -------------- | ------ | ----------------------------------------- | ------- | ------- |
| `text`         | String | The text content displayed in the box     | ""      | false   |

## Events

This component does not emit any custom events.

## Usage

```html
<info-box text="Hello from InfoBox!"></info-box>
```
