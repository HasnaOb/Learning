# HelloWord component

This component renders a screen containing a green box with text displayed inside it.

## Properties

Component can be configured via these properties:

| Name           | Type   | Description                        | Default |
| -------------- | ------ | ---------------------------------- | ------- |
| `Translations` | Object | The translations are stored inside | null    |

## Events

This component has no events.

## Usage

```html
<info-box></info-box>
```
